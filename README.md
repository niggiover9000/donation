# Donation Tool

This is a 100 % vibe coded donation tool for friends and family.

Features:
- Select up to three organisations the donations go to.
- Social Features to engage people to donate more!
- Admin Panel for all settings.
- Music button (music not included due to copyright).

# Installation
- Run the docker-compose.yaml
- Create a folder "sounds" in the docker root folder and add songs that play then the user presses the button.
